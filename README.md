# console_game
A very-fun-to-make console game. Originally, there was going to be more functionality, but I was just messing around with it at the end. It currently supports walking around and very basic interaction with items. Popups are the main way to communicate to the player. I believe I made this during the first semester of the Senior year of high school. 

To run, clone the repository and cd into it. Execute it using "cargo run".
